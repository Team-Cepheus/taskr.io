# taskr.io
A minimal Kanban web app

## Installation

*Running the server locally*
Change `frontend/src/config.js` url to `localhost:3001` or use the hosted url and skip to client setup

In the root directory do 
```
npm install 
npm run startdev
```
*Running the client*

`cd frontend/`  then
```
npm install 
npm start
```
